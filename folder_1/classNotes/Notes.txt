Git commands:

->git --version- to check version
->git init- to initialize git and make current directory the root
->git status- to check the current status
->git add- to add components into staging area
->git commit- to save the changes
->git restore- to undo addition
->git log- to generate log of all git processes performed
->git stash- to remove from staging area
->git help- to generate a helping guide for each command
